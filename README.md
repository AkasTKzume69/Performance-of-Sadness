<h1 align="center">Performance of Sadness</h1>

<p align="center">
  <b>Performance of Sadness</b> is an Android optimization module designed to intelligently enhance game performance using lightweight AI-based detection.  
  It automatically applies a custom performance profile when a supported game is running, then restores your normal system configuration once the game is closed.  
  The module also features dynamic Vulkan/OpenGL switching to improve graphics quality and overall smoothness — all while maintaining system stability.
</p>

<h2 align="center">🎮 What This Module Does</h2>

### 🔍 Game Detection  
- Monitors active processes to detect when games from your configurable game list are launched or closed.  
- Automatically triggers optimizations only when needed.

### 🎨 Vulkan / OpenGL Dynamic Switching  
- Switches to **Vulkan** renderer when a supported game is running for smoother graphics performance.  
- Restores **OpenGL** when no games are active to ensure compatibility and stability.

### 🚀 Performance Profile Application  
- Applies a customized gaming performance profile that optimizes CPU, GPU, I/O, and memory behavior.  
- Reverts back to default system profiles instantly after the game closes.

### 🛠️ System Optimization Tweaks  
Includes tuned adjustments for:  
- Touch responsiveness  
- Scheduler priorities  
- Memory management  
- Smoothness and stability improvements
